# Как создать Pull Request

Чтобы создать Pull Request в реальном репозитории, выполните следующие шаги:

## Вариант 1: Через GitHub Web Interface
1. Перейдите на страницу репозитория на GitHub
2. Выберите вашу текущую ветку (qwen-code-6887824f-3858-4a57-beaa-666362aa1802)
3. Нажмите кнопку "Compare & pull request"
4. Заполните заголовок и описание (можно использовать содержимое файла pull_request_description.md)
5. Назначьте ревьюверов, если необходимо
6. Нажмите "Create pull request"

## Вариант 2: Через GitHub CLI
```bash
gh pr create --title "Редактирование main.py с добавлением комментария" --body "Добавлен комментарий для тестирования изменений в main.py" --base main --head qwen-code-6887824f-3858-4a57-beaa-666362aa1802
```

## Информация о текущих изменениях
- Ветка: qwen-code-6887824f-3858-4a57-beaa-666362aa1802
- Базовая ветка: main
- Измененные файлы: main.py
- Тип изменений: добавление комментария и вывода сообщения об инициализации
